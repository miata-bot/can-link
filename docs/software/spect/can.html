    can-id

     CAN ID

    CAN IDData
    0x69command::8, payload::56

    Command is 1 byte, the payload can take up the remaining 7 bytes.

    commands

     Commands

    CommandNameData
    0x0::8PWRrgb0::1, rgb1::1, addr0::1, addr1::1
    0x1::7, channel::1RGBr::8, g::8, b::8
    0x2::7, channel::1Fadetime_ms:: 16, r::8, g::8, b::8, r::8, g::8, b::8
    0x3::7, channel::1Pixeladdress::16, r::8, g::8, b::8
    0x4::7, channel::1Fillstart::16, end::16, r::8, g::8, b::8
    0x5::7, channel::1Blit--
    0x6::8Modemode::8
    CommandDescription
    PWRcontrols which features are enabled. each feature is represented by a single bit
    RGBSet a RGB channel color. 24 bit color
    FadeFade over time_ms from one 24 bit color to another
    PixelSet a pixel on an addressable channel to a 24 bit color
    FillFill an addressable channel from start to end with a 24 bit color
    BlitShow the current buffer
    Modechange mode. See Modes section
