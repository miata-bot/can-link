    View Source Spect.Network (spect v0.1.0)

    Represents a radio network instance. A network is refered to by it's ID which is a 10 bit integer.

    encryption

     Encryption

    A network can be encrypted at the radio level. This is enabled by having the key set. Packets are encrypted entirely, and the network concept is implemented on top of that, so if a packet is encrypted such that it changes the network id bits, packets will not be routed to the correct network, and also contain nonsense.
